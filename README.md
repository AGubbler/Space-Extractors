# 🚀 Space-Extractors
A Godot-based extraction game where you mine resources and survive the void.

## 🌌 About the Game
Space-Extractors is a project focused on demonstrating an exaggerated reality of global destruction. Built using **Godot 4.6**.

## 🛠️ Getting Started for Contributors
To get this project running on your local machine:
1. **Clone the Repo:** `git clone https://github.com/AGubbler/Space-Extractors`
2. **Open Godot:** Use Godot Engine version **4.6**.
3. **Import:** Click 'Import' in the Project Manager and select the `project.godot` file.

## 🕹️ How to Play
- **WASD or ARROWS:** Move your extractor ship.
- **F3:** Open the Debug Menu
